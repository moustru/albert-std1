Junior Way - Stage 1

Весь код должен быть написан в файле index.js. Там же находятся заготовки для заданий

Чтобы протестировать задания, введи в терминале команду npm test

Задание 1

Написать функцию, которая принимает в качестве аргумента любую строку и возвращает ее же задом наперед

Например: function reverseString('hello') => 'olleh'

Задание 2

Написать функцию, которая принимает в качестве аргумента любое число и возвращает надпись о том, является ли число четным, или нечетным

Число 0 также является четным

Например:

function evenOdd(7) => 'Odd'

function evenOdd(10) => 'Even'

Задание 3

Написать функцию, которая принимает в качестве аргумента любое число и возвращает массив с убывающими числами от этого числа до 1

Если аргумент равен 0, или 1, то функция не выполняется, а появляется предупреждение о неправильном аргументе

Например: function decrementArr(5) => [5, 4, 3, 2, 1]

Задание 4

Написать функцию, которая принимает в качестве аргумента любое число как год и возвращает век, в котором находится этот год

Например:

function getCentury(1705) => 18

function getCentury(1600) => 16

function getCentury(5) => 1

Задание 5

Написать функцию, которая принимает в качестве аргумента массив с числами, возводит каждое число в квадрат и возвращает сумму всех квадратов

Например: function squareSum([3, 2, 5]) => 3^2 + 2^2 + 5^2 = 38

Задание 6

Написать функцию, которая принимает 3 аргумента (все числа) - 2 множителя и длина массива. Функция должна возвращать массив с геометрической прогрессией постепенного перемножения результата каждой итерации на 2 аргумент

Например:

function progression([2, 3, 5]) => [2, 6, 18, 54, 162] - каждое число больше предыдущего в 3 раза и длина массива равна 5

function progression([4, 2, 7]) => [4, 8, 16, 32, 64, 128, 256] - каждое число больше предыдущего в 2 раза и длина массива равна 7
